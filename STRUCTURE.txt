🚀 DÉMARRAGE RAPIDE:

1. MongoDB
   mongod

2. Backend (Nouveau Terminal):
   cd pfe
   node seed.js          # Important! Initialiser les données
   npm start

3. Frontend (Troisième Terminal):
   cd pfe/frontend
   npm install          # Si pas déjà fait
   npm start

4. Accès:
   Frontend: http://localhost:3000
   Backend:  http://localhost:5000
